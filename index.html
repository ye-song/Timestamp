Timestamp Microservice

Example Usage:

Key in the date in the url and it will return the output format.
The following formats are accepted:

  • Hyphenated Date:   https://ye-song.github.io/timestamp/api/2015-12-25
  • String Date:    https://ye-song.github.io/timestamp/api/25 December 2015
  • Unix Time:     https://ye-song.github.io/timestamp/api/1451001600000

Example Output:

unix: 1451001600000
utc: Fri, 25 Dec 2015 00:00:00 GMT
